Desktop App Features

Each feature block maps directly to the requested Swing-only prototype.
01

CardLayout Navigation

LoginPage, HomePage, EditorPage, and PrintPage are framed as one clear desktop flow driven by CardLayout page changes.

02

Dark Editor Workspace

The main editor uses a dark software-style shell with a top command bar, centered button canvas, right inspector, and bottom layer tabs.

03

Live Layer Editing

Text, font, color, rotate, stretch, transparency, and layer visibility update the circular preview in real time.

04

Photo Upload and Crop

The photo layer supports upload, crop selection, direct drag movement inside the button preview, and centering controls.

05

Undo and Redo History

Text edits, slider changes, visibility toggles, image movement, and crop updates are tracked through history.

06

Print and PDF Export

The print screen filters eligible layers, supports drag and drop placement, and exports a working PDF with PDFBox.

System Highlights

Four-Screen Flow

Mock login reaches Home, Start New Project opens Editor, Print completes the workflow, and Log Out returns to Login.

Core 02

Accurate Layer Rules

Hidden layers and layers with zero transparency stay out of the print gallery exactly as required in the editor logic.

Core 03

Paper and Button Options

Print layout responds to the chosen button size while users switch between A4 or Letter and optionally show cut lines.
